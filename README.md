# Paleoclimate Model Comparison


## How to use:

1. Preparation:

- Create ~800gb free disk space
- Install conda
- Run `conda env create -f environment.yml`
- PalMod2 requires a WDCC account. Create an account and write your credentials to `PalMod2_credentials.txt` in the notebook directory:
    - Line 1: username
    - Line 2: password

2. Run data_downloader.ipynb

3. Run data_processer.ipynb

4. Run data_exploration.ipynb
